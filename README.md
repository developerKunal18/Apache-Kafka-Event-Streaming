# Apache Kafka Event Streaming

A Python project demonstrating asynchronous event streaming using Apache Kafka.

## Features

- Kafka producer
- Kafka consumer
- JSON event publishing
- Dockerized Kafka setup

## Technologies Used

- Python
- Apache Kafka
- kafka-python
- Docker

## Installation

```bash
pip install -r requirements.txt
```
Start Kafka:
docker compose up -d

Run Consumer:
python consumer.py

Run Producer:
python producer.py
